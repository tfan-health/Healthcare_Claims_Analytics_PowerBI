# UM_Prior_Auth_PowerBI
Healthcare UM and Prior Auth analytics project

# Utilization Management Dashboard

## Project Overview
Built a Power BI dashboard to analyze healthcare authorization data.

## Business Questions
- How many authorizations were processed?
- What is the approval rate?
- How long do decisions take?
- Which procedures drive authorization volume?

## Key Metrics
- Approval rate: 74.2%
- Average decision time: 6.4 days

## Insights
- Approval rate shows most requests are approved.
- Decision time varies across procedures, indicating workflow differences.
- Certain procedures drive higher authorization volume.

## Tools
- Power BI
- DAX
